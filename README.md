# react-thecatapi-explorer

## Technologies

Project is created with:

- React
- Typescript
- SASS
- Axios
- TheCatAPI API
- react-hook-form
- react-toastify

## Setup

To run this project, install it locally using npm:

```
$ cd ../react-thecatapi-explorer
$ npm install
$ npm start

```
